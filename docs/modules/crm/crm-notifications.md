# Настройка уведомлений сотрудникам

Настройка уведомлений располагается в пункте меню **«Уведомления»**.

В правой части экрана отображаются виды уведомлений для продавцов. При переключении между ними появляется возможность
настроить получателей и дополнительные параметры для каждого типа уведомлений.

---

## Добавление получателя уведомлений

При добавлении получателя в таблицу необходимо указать следующие параметры:

- **Способ получения** – отметка способа доставки уведомления:
    - через Telegram
    - через электронную почту

- **Адрес электронной почты** – указывается, если выбран способ уведомления через электронную почту.

- **Магазин(ы)** – список магазинов, по заказам из которых сотрудник будет получать уведомления.

При этом уведомления о заказах из других магазинов данному сотруднику поступать не будут.
